## 哐哐牌灵感生成机
随机元素生成器。

**玩法**：随机生成一个元素组合，也许帮助绘画/写作/构建故事的灵感。主要是绘画灵感帮助。
比如生成：「绣球花+邮票+木星」这样的随机元素组合。

- **可以收藏，拉黑元素**（有单个元素和元素组合的收藏夹，可以自己创建多个不同的收藏夹）
- **可以自定义词库**（能复制，移动，批量选中，收藏，拉黑，删除。能够只查看收藏拉黑项目）
- **内置8760元素系统词库+筛选版emoji词库**。
- **多种内置随机元素玩法**，增加仪式感（25个随机玩法，自行探索，点击工具箱打开，拖动自行修改顺序）
- **多个内置美化效果，可自定义**（但效果随机）
- **可以导入导出存档备份**（需要手动），建议定期进行
- **netlify部署为网页版**，个人存档会自动保存在浏览器内，但为防止数据丢失，仍然建议手动进行备份

游玩前需要输入口令，且只需要输入一次。但初始化之后仍然需要再次输入。
口令为神秘的5个字，如果是正常途径会知道口令内容。可以祝我和我oc99.
